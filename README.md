YouTube Data API を使用して動画を検索・再生できるアプリね。
だけど、本来の目的は、学校の制限を突破してつまらない授業でYoutube見るためだ。

Node.js 18以上（推奨: 20以上）
npm
YouTube Data API v3 の APIキー

Express をインストール
npm init -y
npm install express

const API_KEY = 'ここAPIキー！！！！！！！';っていう行探して、そこにYouTube Data API のっけてね。

node server.jsで実行ね。

run...
http://localhost:3000
